# Customer Management System

This repository is created to build a Customer Management System.

The project is currently at the initial stage and will be developed incrementally following good software engineering practices.

---

## Tech Stack

**Backend**
- Java 8
- Spring Boot
- Maven
- JUnit
- MariaDB

**Frontend**
- React.js
- Axios

---

## Summary

The system will allow users to manage customer information, including creating, updating, and viewing customers, as well as performing bulk customer operations using Excel files.

Implementation, database scripts, and detailed documentation will be added as development progresses.
